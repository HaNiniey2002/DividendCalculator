A simple Android app to calculate dividends based on investment amount, annual dividend rate, and number of months invested.


## Features
* Accepts Invested Fund Amount, Annual Dividend Rate (%), and Number of Months Invested (Max 12 months).
* Calculates Monthly Dividend: `(Rate / 12) x Invested Fund`.
* Calculates Total Dividend: `Monthly Dividend x Number of Months`.
* Displays results clearly with proper formatting (2 decimal places).
* Includes a navigation menu with Home and About pages.
* The About Page displays the Application Icon, Author Information, Copyright Notice, and a clickable GitHub Repository URL.

## How to Run
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/HaNiniey2002/DividendCalculator.git](https://github.com/HaNiniey2002/DividendCalculator.git)
    ```
2.  **Open in Android Studio:**
    * Launch Android Studio.
    * Select "Open an existing Android Studio project" and navigate to the cloned `DividendCalculator` directory.
3.  **Sync Gradle:**
    * Once the project is open, Android Studio should automatically sync Gradle. If not, click "Sync Project with Gradle Files" from the toolbar.
4.  **Run on an Emulator or Device:**
    * Select a virtual device (emulator) or connect a physical Android device.
    * Click the "Run 'app'" button (green play icon) in the toolbar.
